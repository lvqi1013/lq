# 图片资源使用说明

## 个人照片更换方法

1. 准备一张您的个人照片，建议尺寸：400x400像素，JPG或PNG格式
2. 将照片命名为 `profile.jpg`
3. 替换此文件夹中的 `profile.jpg` 文件（如果不存在则直接放入）

## 其他图片资源

您可以根据需要在此文件夹中添加其他图片资源，例如：
- 项目截图
- 获奖证书照片
- 技术栈图标

## 图片格式建议

- 使用压缩后的图片格式（如优化后的JPG、WebP）
- 控制图片大小，单个图片建议不超过2MB
- 对于图标，优先使用SVG格式以保证清晰度

## 注意事项

- 确保所有图片资源的版权问题
- 个人照片请使用正式、专业的形象照
- 敏感信息（如证件号码）请适当处理后再上传
