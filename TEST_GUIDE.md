# 浏览器兼容性测试指南

## 如何测试

1. **本地打开测试**
   - 直接双击 `index.html` 文件在默认浏览器中打开
   - 或右键点击文件，选择「打开方式」并选择不同的浏览器

2. **使用本地服务器测试**（推荐）
   - 在 `AI_Resume` 文件夹中打开命令提示符
   - 运行以下命令启动简单HTTP服务器：
     - Python 2.x: `python -m SimpleHTTPServer 8000`
     - Python 3.x: `python -m http.server 8000`
     - Node.js: `npx http-server -p 8000`
   - 然后在浏览器中访问 `http://localhost:8000`

## 测试浏览器列表

建议在以下浏览器中进行测试，确保最佳兼容性：

- Chrome (最新2个版本)
- Firefox (最新2个版本)
- Safari (最新2个版本)
- Edge (最新2个版本)
- 移动设备浏览器（如iPhone Safari、Android Chrome）

## 测试要点

1. **布局显示**
   - 确认页面布局是否正确显示
   - 检查元素对齐和间距是否合理
   - 验证导航栏和各部分是否正常显示

2. **响应式设计**
   - 调整浏览器窗口大小，检查页面是否正确响应
   - 在不同设备（手机、平板、电脑）上测试
   - 验证汉堡菜单在小屏幕上是否正常工作

3. **交互功能**
   - 测试导航栏点击跳转是否正常
   - 验证技能条动画是否正常显示
   - 检查表单验证功能
   - 测试返回顶部按钮功能

4. **图片显示**
   - 确认个人照片是否正确显示
   - 检查图片加载性能

5. **其他检查**
   - 确认所有文字清晰可读
   - 检查颜色对比度是否合适
   - 验证所有链接是否正常工作

## 常见问题及解决方法

1. **图片不显示**
   - 检查图片路径是否正确
   - 确认图片文件格式和大小是否合适

2. **样式异常**
   - 清除浏览器缓存后重试
   - 检查是否存在CSS兼容性问题

3. **JavaScript错误**
   - 打开浏览器开发者工具（F12）查看控制台错误
   - 根据错误信息修复问题

## 优化建议

- 根据测试结果，调整响应式断点
- 优化图片大小，提高加载速度
- 根据不同浏览器特性调整样式和交互
